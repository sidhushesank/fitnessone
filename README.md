# Fitness One LB Nagar – Website

Official website project for **Fitness One 2nd Branch, LB Nagar**, a large multi-activity fitness facility in Hyderabad offering gym training, combat sports, swimming, and sports programs under one roof.

🔗 **Live Website:** https://fitnessone.vercel.app/

---

# Project Overview

This project is a modern web representation of **Fitness One – LB Nagar Branch**, designed to showcase the gym's infrastructure, facilities, and fitness programs.

The website highlights the scale of the facility, available activities, and provides essential information for visitors looking to join or learn more about the gym.

The goal of this project is to create a **clean, fast, and visually engaging website experience** for a premium fitness environment.

• Indoor swimming pool
• Sauna facilities
• Combat sports training areas
• Box cricket arena
• Cricket academy training space

The facility supports training for **fitness enthusiasts, athletes, and beginners**.
